# fitness-assistant-voice

Пример голосового ассистента на базе Realtime API Yandex AI Studio.  
Проект демонстрирует базовый сценарий голосового взаимодействия: распознавание речи, генерация ответа, вызов функций и синтез речи.

Ассистент работает в фитнес-домене и может:

- рассчитать ориентировочную норму калорий и белка;
- предложить базовые спортивные добавки;
- искать информацию в локальной базе знаний через `file_search`;
- искать актуальные материалы в интернете через `web_search`;
- корректно завершать диалог.

Проект предназначен для демонстрации возможностей Realtime API и может быть адаптирован под другие сценарии.

## Возможности примера

В приложении реализованы:

- потоковое распознавание речи;
- генерация ответа моделью в реальном времени;
- голосовой ответ;
- вызов пользовательских функций;
- использование встроенных инструментов `file_search` и `web_search`;
- обработка завершения диалога.

## Требования

- Python 3.10+
- доступ к Yandex Cloud
- API-ключ
- ID каталога
- микрофон и наушники для локального запуска

## Установка

Создайте виртуальное окружение и установите зависимости:

```bash
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

pip install yandex-ai-studio-sdk sounddevice numpy aiohttp
```

## Настройка переменных окружения

Перед запуском задайте переменные окружения:

```bash
export YANDEX_CLOUD_FOLDER_ID="your_folder_id"
export YANDEX_CLOUD_API_KEY="your_api_key"
```

Если планируется использование `file_search`, дополнительно укажите ID поискового индекса:

```bash
export VECTOR_STORE_ID="your_vector_store_id"
```

## Настройка базы знаний

Чтобы использовать поиск по локальным материалам:

1. создайте индекс в Yandex AI Studio;
2. загрузите в него файлы из папки [data](https://github.com/yandex-ai-studio/fitness-assistant/tree/main/data);
3. передайте ID индекса через `VECTOR_STORE_ID`.

Этот шаг можно пропустить, если поиск по базе знаний не требуется.

## Запуск

```bash
source venv/bin/activate
python voice_agent.py
```

После запуска ассистент начнёт прослушивать микрофон и отвечать голосом.

## Примеры запросов

Расчёт калорий:

- «Помоги рассчитать калории»
- «Вес 70 кг, рост 175 см, возраст 25 лет, тренируюсь 3 раза в неделю»

Добавки:

- «Что можно принимать для набора массы?»
- «Я новичок»
- «Что рекомендуют для выносливости?»

Общие вопросы:

- «С чего начать тренировки?»
- «Как часто тренироваться новичку?»

## Примечания

- Для корректной работы рекомендуется использовать наушники.
- Ответ ассистента синтезируется голосом и может попасть в распознавание при использовании динамиков.
- Пример предназначен для демонстрации и экспериментов и не является production-решением.
