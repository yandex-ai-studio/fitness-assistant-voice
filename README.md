# fitness-assistant-voice

# 🏋️ Голосовой Фитнес-Ассистент

Умный голосовой ассистент для помощи в тренировках, питании и выборе спортивных добавок на базе Yandex Realtime API.

## ⚠️ ВАЖНЫЙ ДИСКЛЕЙМЕР

**Это демонстрационный пример использования Yandex Realtime API.**

- Все рекомендации по питанию, тренировкам и добавкам носят исключительно **демонстрационный характер**
- Данный пример **НЕ предназначен** для использования в реальных медицинских или фитнес-приложениях
- Перед началом любых тренировок или изменением рациона **обязательно проконсультируйтесь с врачом**
- Разработчики **не несут ответственности** за использование информации из этого примера
- Цель проекта — показать возможности голосового API, а не предоставить профессиональные консультации

**Используйте этот код только для обучения и экспериментов с технологией!**

## 🎯 Возможности

### 1. Расчет калорий и макронутриентов
Ассистент рассчитывает:
- Базовый метаболизм (BMR)
- Рекомендуемое потребление калорий
- Калории для похудения (дефицит 500 ккал)
- Калории для набора массы (профицит 500 ккал)
- Рекомендуемое количество белка

**Пример диалога:**
```
Пользователь: "Рассчитай мне калории"
Ассистент: "Конечно! Скажите ваш вес, рост, возраст, пол и уровень активности"
Пользователь: "75 кг, 180 см, 28 лет, мужчина, тренируюсь 4 раза в неделю"
Ассистент: [вызывает функцию и озвучивает результат]
```

### 2. Рекомендации по спортивным добавкам
Подбирает добавки в зависимости от:
- Цели (набор массы / похудение / выносливость)
- Уровня опыта (новичок / средний / продвинутый)

**Пример диалога:**
```
Пользователь: "Какие добавки мне принимать для набора массы?"
Ассистент: "Какой у вас опыт тренировок?"
Пользователь: "Я новичок"
Ассистент: [рекомендует креатин, протеин, витамины с дозировками]
```

### 3. База знаний о фитнесе
Через функцию `file_search` ассистент может найти информацию о:
- Программах тренировок для новичков
- Ответах на частые вопросы о спортзале
- Ведении дневника тренировок
- Спортивных добавках (122 позиции с рейтингами)

**Пример диалога:**
```
Пользователь: "С чего начать в спортзале?"
Ассистент: [ищет в базе знаний и дает рекомендации]
```

### 4. Поиск актуальной информации
Через `web_search` находит свежую информацию о:
- Новых методиках тренировок
- Научных исследованиях
- Трендах в фитнесе

## 🚀 Установка и запуск

### 1. Установите зависимости

```bash
# Создайте виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate

# Установите пакеты
pip install aiohttp numpy sounddevice
pip install git+https://github.com/yandex-cloud/yandex-cloud-ml-sdk.git@experimental_mic
```

### 2. Настройте credentials

Откройте `voice_agent.py` и укажите:
```python
YANDEX_CLOUD_FOLDER_ID = "ваш_folder_id"
YANDEX_CLOUD_API_KEY = "ваш_api_key"
```

### 3. (Опционально) Настройте базу знаний

Если хотите использовать функцию `file_search`:
1. Создайте индекс в Yandex AI Studio
2. Загрузите файлы из `data/` проекта [fitness-assistant](https://github.com/yandex-datasphere/fitness-assistant)
3. Укажите ID индекса:
```python
VECTOR_STORE_ID = "ваш_vector_store_id"
```

### 4. Запустите ассистента

```bash
source venv/bin/activate
cd realtime
python voice_agent.py
```

## ⚠️ Важные замечания

1. **Используйте наушники!** Иначе звук синтеза речи будет активировать распознавание
2. **Микрофон:** Убедитесь, что микрофон подключен и работает
3. **Интернет:** Требуется стабильное подключение к интернету
4. **Консультация врача:** Ассистент напоминает о необходимости консультации перед приемом добавок

## 🔧 Технические детали

### Функции ассистента

#### `calculate_calories(weight, height, age, gender, activity_level)`
- Использует формулу Миффлина-Сан Жеора
- Учитывает уровень активности
- Возвращает BMR, дневные калории, рекомендации по белку

#### `recommend_supplements(goal, experience)`
- База из 122 спортивных добавок
- Рейтинги эффективности (1-5 звезд)
- Дозировки и назначение

### Параметры аудио
- Частота дискретизации: 44100 Hz
- Формат: PCM16
- Голос: "dasha"
- VAD: серверный (порог 0.5, тишина 400мс)

## 🐛 Решение проблем

### Ошибка "ModuleNotFoundError: yandex_cloud_ml_sdk._experimental"
```bash
pip install git+https://github.com/yandex-cloud/yandex-cloud-ml-sdk.git@experimental_mic
```

### Ошибка "UNAUTHENTICATED"
Проверьте правильность API ключа и folder ID

### Микрофон не работает
- Проверьте права доступа к микрофону
- Убедитесь, что sounddevice установлен корректно
- Попробуйте другой микрофон

### Эхо или петля обратной связи
Используйте наушники!
